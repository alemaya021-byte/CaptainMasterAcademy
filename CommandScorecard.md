# Command Performance Scorecard

## Purpose

The Command Performance Scorecard summarizes command ability from completed incident simulations.

## Graded Domains

The scorecard grades:

- Incident Command
- Tactical Decisions
- Communications
- Safety
- Leadership
- Accountability
- Resource Management
- Risk Management

## Scoring Model

Incident Command uses the overall incident score.

Tactical Decisions, Communications, Safety, and Leadership use the corresponding incident simulator domain scores.

Accountability blends safety, leadership, and communications. Resource Management blends tactical decisions and leadership. Risk Management blends safety and tactical decisions.

## Historical Improvement

Historical improvement compares earlier incident attempts against later incident attempts and displays the point change for each domain.

## Data Source

The scorecard uses `CMA.incidentAttempts()`, which merges local incident history and synced incident reports.
