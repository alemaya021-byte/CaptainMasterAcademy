# Decision Analysis

## Question Decision Analysis

For missed quiz and exam questions, the mentor analyzes:

- Selected answer
- Correct answer
- Source-backed rationale
- Incorrect-answer explanation, when available
- Source reference
- Related questions
- Policy/SOP reference classification
- Test-writer trap pattern
- Memory strategy

The mentor identifies common promotional-exam traps such as:

- EXCEPT/NOT inversion
- Timeline drift
- Role swap
- Mandatory-language changes
- Fireground sequence errors
- Source-wording changes

## Incident Decision Analysis

For incident simulations, every tactical decision phase is reviewed:

1. Initial size-up, radio report, and command mode
2. Tactical priorities, resources, accountability, and RIT
3. Search, fire attack, and water supply
4. Ventilation, exposure protection, and safety
5. Mayday management
6. Demobilization and transfer/closeout

Each phase analysis includes:

- What happened
- Why it was incorrect or why it aligned with best practice
- Better decision
- Best decision
- Tactical considerations
- Safety considerations
- Leadership considerations
- Communication considerations
- Relevant department policy
- Relevant SOP or operational reference
- Related questions
- Similar historical mistakes
- Memory strategy

## Learning Profile Impact

Each review updates the Chief Mentor profile:

- Mastery reflects demonstrated command or source-rule control.
- Confidence reflects decision quality and leadership control.
- Retention reflects whether the topic should come back soon.
- Readiness blends score, safety, communications, and source accuracy.

The profile is intentionally separate from the question database so the source-of-truth question bank remains unchanged.
