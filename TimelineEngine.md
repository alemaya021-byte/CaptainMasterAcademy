# Timeline Engine

The Version 2.5 timeline engine powers the interactive Performance Timeline on `tracking.html`.

## Timeline Filters

The dashboard supports:

- Today
- 7 Days
- 30 Days
- 90 Days
- All Time

## Timeline Metrics

The timeline displays:

- Readiness
- Mastery
- Confidence
- Retention
- Practice exam scores
- Incident simulator scores
- Study hours
- Questions completed
- Streak activity

## Data Sources

The timeline reads from:

- `progress.daily`
- `progress.adaptive.predictions.snapshots`
- `progress.exams`
- `progress.incidents`
- Synced incident reports
- Existing adaptive trend data

## Mobile Behavior

The timeline is horizontally scrollable on small screens and uses stable table widths so phone, tablet, and desktop layouts remain readable.
